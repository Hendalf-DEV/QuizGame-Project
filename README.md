# QuizGame Project

End project for React course at Omnia
<br>
Reaktin loppuprojekti Omniassa.

# Description
A simple quiz game application built with React. Users can answer multiple-choice questions and receive a score at the end of the quiz.
<br>
Yksinkertainen tietovisa-sovellus, joka on rakennettu Reactilla. Käyttäjät voivat vastata monivalintakysymyksiin ja saada pisteet visan lopussa.

# Features

- Multiple choice questions / monivalintakysymykset
- Score calculation / pisteiden laskenta
- Highscore list / pisteiden lista
- Responsive design / responsiivinen ulkoasu
- MognoDB database / MongoDB-tietokanta
